# In Harm’s Way Page Plan

## Project

Eric Deloach Author Website

## Purpose of This Document

This document defines the content plan, page structure, reader journey, character feature strategy, media strategy, retailer strategy, metadata direction, schema needs, and conversion goals for the **In Harm’s Way** book page.

Claude Code must follow this plan when building the *In Harm’s Way* page and must not rewrite approved copy unless the user explicitly requests revisions.

## Approved Page URL


## Page Purpose

The *In Harm’s Way* page should introduce the book as Eric Deloach’s grounded action and military thriller lane.

This page should feel tactical, intense, character-driven, cinematic, and grounded. It should focus on brotherhood, survival, Navy personnel, tactical danger, violence, consequence, revenge, and the dangerous world of Charlie One Alpha.
