# Media and Social Plan

## Project

Eric Deloach Author Website

## Purpose of This Document

This document defines the media and social strategy for the Eric Deloach author website.

It covers YouTube trailers, YouTube Shorts, TikTok, Instagram, Facebook, Goodreads, the current Myth and Mayhem website, media embeds, social links, content rules, accessibility requirements, and verification requirements.

Claude Code must follow this document when implementing media sections, social links, video embeds, and media-related pages.

The media strategy should help visitors:

1. Watch book trailers
2. Discover Eric’s YouTube presence
3. Explore character-focused shorts
4. Follow Eric on social platforms
5. Connect with the Myth and Mayhem brand
6. Move from media content to book pages and retailer links

If a video URL is missing, do not render an embed or public video card.

```text

```

Do not display:

```text

Trailer placeholder

TODO
```

unless the client explicitly approves that public message.

## Copyright and Embed Rules

Do not copy long social captions or video descriptions into the website unless approved.

Do not scrape and republish social media content beyond normal embeds or approved excerpts.

Do not use platform logos in a way that violates brand guidelines.

Do not download and host YouTube thumbnails unless approved and allowed.

## Media Accessibility Rules

All video sections should include text context.

Recommended accessible structure:

```text
Section heading
Short description
Responsive video embed
Text CTA to related book page
Text CTA to YouTube channel
```

Each iframe should have a clear title, such as:

```text
Rage of Angels trailer by Eric Deloach
```

or:

```text
In Harm’s Way promotional trailer by Eric Deloach
```

Do not use generic iframe titles like:

```text
Video
YouTube
Embed
```

## Performance Rules

Video embeds can slow down pages.

Recommended approach:

```text
Use lazy loading for embeds when possible
Consider thumbnail-first embed pattern
Limit homepage embeds to the highest-priority videos
Move larger video grids to the Media page
Avoid embedding too many Shorts on the homepage
```

## Homepage Video Count Recommendation

Homepage:
