# SEO and AEO Plan

## Project

Eric Deloach Author Website

## Purpose of This Document

This document defines the SEO and AEO strategy for the Eric Deloach author website.

The goal is to help the website appear clearly in search engines, AI search results, rich results, featured snippets, and reader discovery paths for Eric Deloach, his books, his author brand, and his core genres.

This document should guide page structure, metadata, headings, schema, internal linking, FAQs, image alt text, and content organization.

## Primary SEO Goal

The website should become the main author-owned source for Eric Deloach’s books, brand, media, and reader pathways.

The site should support discovery for:


## Content Depth Strategy

Minimum content goals:


## Duplicate Content Rule

Do not copy the exact same long synopsis across multiple pages unless it is intentionally locked.

Use:

```text
Short teaser on homepage
Medium overview on Books page
Full approved copy on individual book page
Series-focused explanation on series page
```

Preserve approved copy when locked, but avoid unnecessary duplication in new sections.
